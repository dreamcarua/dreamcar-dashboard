# 📊 UTM Dashboard - Аналитика лидов из SendPulse

Полнофункциональный дашборд для анализа UTM-меток из контактов SendPulse.

## 🎯 Возможности

- ✅ Автоматическая синхронизация контактов из SendPulse
- ✅ Интеграция с Google Sheets для хранения данных
- ✅ Очистка и нормализация UTM-меток
- ✅ Визуализация данных через графики (Chart.js)
- ✅ Одностраничный интерфейс без перезагрузок
- ✅ Telegram уведомления о важных событиях
- ✅ Система логирования всех операций

## ⚙️ Настройка

### 1. SendPulse API

Откройте `config/app_config.php` и замените:

```php
define('SENDPULSE_ID', 'YOUR_SENDPULSE_ID');
define('SENDPULSE_SECRET', 'YOUR_SENDPULSE_SECRET');
```

### 2. Google Sheets API

1. Создайте Service Account в Google Cloud Console
2. Скачайте JSON с учетными данными
3. Сохраните как `data/google_credentials.json`
4. В `config/app_config.php` укажите ID таблицы:

```php
define('GOOGLE_SPREADSHEET_ID', 'YOUR_SPREADSHEET_ID');
```

### 3. Telegram бот

Токен и Chat ID уже настроены в `app_config.php`:

```php
define('TELEGRAM_BOT_TOKEN', 'BOT_TOKEN_FROM_BOTFATHER');
define('TELEGRAM_CHAT_ID', '-4800447687');
```

## 🚀 Использование

### Первый запуск

1. Откройте дашборд: `https://dreamcar.ai-platform.space/dashboard/utm-dashboard/`
2. При первом запуске будут созданы **тестовые данные**
3. Нажмите **"🔄 Синхронизировать данные"** для загрузки реальных контактов

### Основные функции

- **📊 Обзор** - общая статистика и графики
- **📍 Источники** - анализ utm_source
- **🔗 Тип трафика** - анализ utm_medium
- **🎯 Кампании** - ТОП кампаний
- **📋 Детали** - utm_term и utm_content
- **📋 Показать JSON** - просмотр сырых данных

## 📊 Аналитика

Дашборд показывает:

1. **Общая статистика**
   - Всего лидов
   - Количество источников
   - Количество кампаний
   - Лиды сегодня

2. **Графики**
   - Лиды по дням (линейный)
   - Распределение по источникам (пончик)
   - ТОП источников (горизонтальный бар)
   - Типы трафика (пирог)

3. **Таблицы**
   - Источники с долей и трендом
   - Типы трафика с конверсией
   - ТОП-20 кампаний
   - Ключевые слова
   - Варианты объявлений

## 🔄 Автоматизация

### Настройка cron для автоматической синхронизации

Добавьте в crontab:

```bash
# Синхронизация каждый час
0 * * * * curl -X POST https://dreamcar.ai-platform.space/dashboard/utm-dashboard/api/handler.php?action=sync
```

### Или через Make/Zapier

1. Создайте HTTP запрос к `api/handler.php?action=sync`
2. Метод: POST
3. Расписание: каждый час

## 📝 Логирование

Все события системы логируются в `log_actual.json`:

- ✅ Успешные операции
- ⚠️ Предупреждения
- ❌ Ошибки
- 📊 Статистика

Важные события отправляются в Telegram.
